E-commerce Api

E-commerce Api is restful api that designed to handle users, products

, carts and orders features using node.js and mongoDb.

Users Endpoint :

get all users > GET : api/users

get user by id > GET : api/users/:id

register a user > POST : api/users/register

user login > POST : api/users/login

user logout > POST : api/users/logout

user logout all devices > POST : api/users/logoutAll

user account > GET : api/users/me

user update his account > PUT : api/users/me

user delete his acount > DELETE : api/users/me

Products Endpoint :

get all products > GET : /api/product

add products > POST : /api/product

get product by id > GET : /api/product/:id

update product > PATCH : /api/product/:id

delete product > DELETE : /api/product/:id

Orders Endpoint :

get all orders > GET : /api/orders

create order > POST : /api/orders

get order by id > GET : /api/orders/:id

update order > PATCH : /api/orders/:id

delete order > DELETE : /api/orders/:id

get all user order > GET : /api/orders/myorders

Cart Endpoint :

get all my carts > GET : /api/cart

create cart > POST : /api/cart

get cart by id > GET : /api/cart/:cart-id

delete cart > DELETE : /api/cart/:cart-id

get all products in my cart > GET : /api/cart-id/products

add product to cart > PATCH : /api/cart/:cart-id/:product-id

delete product from cart > DELETE : /api/cart/:cart-id/:product-id

get all products status from my cart > GET : /api/cart-id/products/status

get product status in my cart > GET : /api/cart-id/product-id/status
